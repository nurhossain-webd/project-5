# 🐞 GitHub Issue Tracker

## Overview
GitHub Issue Tracker is a web application that allows users to log in with demo credentials, browse issues by category, search issues, and view full issue details in a modal. The project is built with a responsive layout and interactive filtering features.

## Live Link
https://github-issues-tracker-by-nur.netlify.app/

## Technologies Used
- HTML
- CSS
- Tailwind CSS
- DaisyUI
- JavaScript

## Main Features
- Login with demo credentials
- Filter issues by All, Open, and Closed tabs
- Search functionality for issues
- Modal view for full issue details
- Responsive issue card layout with loading spinner

## Dependencies
- Tailwind CSS
- DaisyUI
- JavaScript
- REST API integration
